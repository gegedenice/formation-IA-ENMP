# Istex TDM

[Istex](https://www.istex.fr/) est une infrastructure de service pour la fouille de texte s'appuyant sur la mise à disposition de corpus (TI + métadonnées enrichies et modélisées) issues des programmes d'acquisitions des Licences Nationales. Istex se décline en plusieurs services :

| Service                                         | Accès                                                                                                          | Remarque                                                                                                                                                      |
| ----------------------------------------------- | -------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Site Licences Nationales                        | [https://www.licencesnationales.fr/](https://www.licencesnationales.fr/)                                       | <ul><li>Abes : contrats &#x26; signalement</li><li>Couperin + Gis Collex-Persée : choix des corpus à acquérir</li><li>Inist-CNRS : plateforme Istex</li></ul> |
| Site web Istex                                  | [https://www.istex.fr/](https://www.istex.fr/)                                                                 |                                                                                                                                                               |
| Documentation                                   | [https://doc.istex.fr/](https://doc.istex.fr/)                                                                 |                                                                                                                                                               |
| Moteur de recherche                             | [Istex Search](https://search.istex.fr/fr-FR)                                                                  | Avec téléchargement du corpus constitué dans des formats compatibles avec des outils de fouille de textes (Lodex, Cortext, Gargantext...)                     |
| Revues de sommaires                             | [https://revue-sommaire.istex.fr/](https://revue-sommaire.istex.fr/)                                           | <p>Pour chaque revue Istex : </p><ul><li>accès par année - volume - numéro - titre</li><li>Export KBART des états de collection </li></ul>                    |

* [Loterre](https://loterre.istex.fr/fr/) (Linked open terminology resources) : une plateforme d’exposition et de partage de terminologies scientifiques modélisées selon les principes du web sémantique&#x20;
* [Web services TDM](https://services.istex.fr/category/services/) : collection de traitements NLP prêts à l'emploi mis à disposition par API.&#x20;
  * Les web services ISTEX TDM sont des modules d’analyse textuelle permettant d'appliquer des traitements de NLP (extraction de termes, validation de références, reconnaissance d’entités, etc.) sur des corpus de documents ou des métadonnées.&#x20;
  * Chaque service est décrit sur [https://services.istex.fr/](https://services.istex.fr/) (par exemple [https://services.istex.fr/calcul-de-similarite-entre-les-documents-dun-corpus/](https://services.istex.fr/calcul-de-similarite-entre-les-documents-dun-corpus/)) et dispose d’une documentation Swagger : un formulaire web pour tester directement une requête (“Try it out”) et voir la réponse JSON sans coder. C’est la porte d’entrée “technique” pour explorer la richesse des services ISTEX.
* [TDM factory](https://tdm-factory.services.istex.fr/)  : interface d'exécution des web services TDM sur ses propres données.&#x20;
  * La TDM Factory est une interface en ligne qui facilite l’utilisation des web services sans avoir à écrire de code. L’utilisateur charge un fichier (CSV, TSV, JSON), choisit un ou plusieurs services ISTEX TDM à appliquer, et reçoit les résultats par e-mail sous forme de fichier enrichi.&#x20;
  * Tous les Web services TDM ne sont utilisables dans la Factory
  * C’est la solution la plus simple pour expérimenter avec ses propres données, même si le système reste un peu “boîte noire” (pipeline interne, erreurs peu explicites, délai de traitement par mail).
* [Lodex](https://www.lodex.fr/) : application de dashboarding permettant de convertir des données structurées en site web interactif de data visualisation. **LODEX** est un outil open source de publication et de valorisation de données tabulaires. Il permet d’importer un tableur ou un CSV, d’enrichir les données grâce à des connecteurs de web services (dont les services ISTEX TDM), puis de proposer des visualisations interactives (graphiques, facettes, cartes, exports).&#x20;
* [TM Tools explorer](https://data.istex.fr/instance/tm-tools-explorer) (basé sur l'ontologie OntoT&#x4D;**)** : catalogue de références d'outils open source de TDM&#x20;
* [Projet IALAC](https://www.istex.fr/ressources-pour-le-fouille-de-textes/#ancre3) (2026) : lac (datalake) de publications, plateforme unifiée de stockage multi-documents garantissant l'accès aux données pour des traitements IA et la tracabiliét des ces traitements
