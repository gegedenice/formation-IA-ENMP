# A retenir

## Modèle IA = algorithme + données

* Un modèle d'IA est la résultante de la combinaison de deux composantes : un algorithme entraîné (ajusté) sur des données pour faire des prédictions -> **tout modèle d'IA (y compris générative) est donc intrinsèquement associé à des données d'entraînement**.
* Lorsque l'algorithme tourne sur des jeux de données locales, le corollaire est que le modèle produit n'est à priori pas partageable en tant que tel.

## Garbage in, garbage out

La qualité et la pertinence des prédictions d'un modèle dépend fondamentalement de celle des données d'entraînement. C'est le principe de base de tout modèle d'IA, valable pour le Machine Learning "simple" comme pour l'IA générative
